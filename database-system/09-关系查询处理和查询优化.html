# 关系代数表达式等价变换规则

连接、笛卡尔积的交换律：

  • 连接的交换律： R⋈S=S⋈RR \bowtie S = S \bowtie RR⋈S=S⋈R
  • 笛卡尔积的交换律： R×S=S×RR \times S = S \times RR×S=S×R

连接、笛卡尔积的结合律：

  • 连接的结合律： (R⋈S)⋈T=R⋈(S⋈T)(R \bowtie S) \bowtie T = R \bowtie (S \bowtie T)(R⋈S)⋈T=R⋈(S⋈T)
  • 笛卡尔积的结合律：(R×S)×T=R×(S×T)(R \times S) \times T = R \times (S \times T)(R×S)×T=R×(S×T)

投影的串接定律：

  • 对于关系 RRR，πA(πB(R))=πA(R)\pi_{A}(\pi_{B}(R)) = \pi_{A}(R)πA​(πB​(R))=πA​(R)，AAA 构成 BBB 的子集。

选择的串接定律：

  • 对于关系 RRR， σP(σQ(R))=σP∧Q(R)\sigma_{P}(\sigma_{Q}(R)) = \sigma_{P \land Q}(R)σP​(σQ​(R))=σP∧Q​(R)

选择与投影操作的交换律：

  • 对于关系 RRR， σP(πA(R))=πA(σP(R))\sigma_{P}(\pi_{A}(R)) = \pi_{A}(\sigma_{P}(R))σP​(πA​(R))=πA​(σP​(R))

选择与笛卡尔积的交换律：

  • 对于关系 RRR 和 SSS， σP(R×S)=(σP(R))×(σP(S))\sigma_{P}(R \times S) = (\sigma_{P}(R)) \times (\sigma_{P}(S))σP​(R×S)=(σP​(R))×(σP​(S))

选择与并的分配律：

  • 对于关系 RRR 和 SSS 和选择条件 PPP， σP(R∪S)=σP(R)∪σP(S)\sigma_{P}(R \cup S) = \sigma_{P}(R) \cup \sigma_{P}(S)σP​(R∪S)=σP​(R)∪σP​(S)

选择与差运算的分配律：

  • 对于关系 RRR 和 SSS， σP(R−S)=σP(R)−σP(S)\sigma_{P}(R - S) = \sigma_{P}(R) - \sigma_{P}(S)σP​(R−S)=σP​(R)−σP​(S)

选择对自然连接的分配律：

  • 对于关系 RRR 和 SSS， σP(R⋈S)=(σP(R))⋈S=R⋈σP(S)\sigma_{P}(R \bowtie S) = (\sigma_{P}(R)) \bowtie S = R \bowtie \sigma_{P}(S)σP​(R⋈S)=(σP​(R))⋈S=R⋈σP​(S)

投影与笛卡尔积的分配律：

  • 对于关系 RRR 和 SSS 和 TTT， πA(R×S)=(πA(R))×(πA(S))\pi_{A}(R \times S) = (\pi_{A}(R)) \times (\pi_{A}(S))πA​(R×S)=(πA​(R))×(πA​(S))

投影与并的分配律：

  • 对于关系 RRR 和 SSS，和属性集合 AAA，πA(R∪S)=πA(R)∪πA(S)\pi_{A}(R \cup S) = \pi_{A}(R) \cup \pi_{A}(S)πA​(R∪S)=πA​(R)∪πA​(S)

# 查询树的启发式优化

典型的启发式规则：

  • 选择运算应尽可能先做，在优化策略中这是最重要、最基本的一条。
  • 把投影运算和选择运算同时进行，避免重复扫描关系。
  • 把投影同其前或其后的双目运算结合起来，没有必要为了去掉某些字段而扫描一遍关系。
  • 把某些选择同在它前面要执行的笛卡尔积结合起来成为一个连接运算，连接特别是等值连接运算要比同样关系上的笛卡尔积省很多时间。
  • 找出公共子表达式：如果这种重复出现的子表达式的结果不是很大的关系，并且从外存中读入这个关系比计算该子表达式的时间少得多，则先计算一次公共子表达式并把结果写入中间文件是合算的。当查询的是视图时，定义视图的表达式就是公共子表达式的情况。
