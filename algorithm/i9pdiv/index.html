# 排序算法

# 插入排序

# 算法描述

对于待排序数组，从第 2 个元素开始（称作插入对象元素）比较它与之前的元素（称作比较对象元素），当插入对象元素小于比较对象元素时，继续往前比较，直到大于等于比较对象，此时将插入对象元素插入到该次比较对象元素之后。重复这个插入过程直到最后一个元素作为插入对象元素完成插入操作。

# 折半插入优化

直接插入排序每次向已排序序列中插入元素时，都要从已排序序列的最后开始逐一比较大小寻找插入元素的合适位置。折半插入排序通过折半查找寻找合适的位置，然后一次性进行移动，为插入元素腾出位置。

折半插入的关键在于找到插入位置，折半过程代码如下。

while (low <= high) {
    int center = low + (high - low) / 2;
    if (arr[center] <= target) low = center + 1;
    else high = center - 1;
}

public int[] insertSort(int[] arr) {
    if (arr.length < 2) return arr;
    for (int i = 1; i < arr.length; i++) {
        int target = arr[i], j = i - 1;
        for (; j >= 0; j--) {

    }

public int[] insertSortBinary(int[] arr) {
    if (arr.length < 2) return arr;
    for (int i = 1; i < arr.length; i++) {
        if (arr[i - 1] <= arr[i]) continue;
        int target = arr[i];
        int low = 0, high = i - 1;
        while (low <= high) {
            int center = low + (high - low) / 2;
            if (arr[center] <= target) low = center + 1;
            else high = center - 1;
        }
        for (int j = i; j > low; j--) {
            arr[j] = arr[j - 1];
        }
        arr[low] = target;
    }
    return arr;
}

# 希尔排序

# 算法描述

希尔排序是简单插入排序的改进，它基于以下事实。

  • 简单插入排序对排序程度较高的序列有较高的效率。假设初始序列已完全排序，则每一轮均只需比较一次，将得到 O(n)O(n)O(n) 的线性复杂度，冒泡排序和选择排序做不到这一点，均仍需 O(n2)O(n^2)O(n2) 次比较。
  • 简单插入排序每次比较最多将数字移动一位，效率较低。

Donald Shell 在 1959 年发表的论文中，针对第二点提出如下方法。对原待排序列中相隔 gapgapgap 的数字执行简单插入排序，然后缩小 gapgapgap，对新的 gapgapgap 间隔的数字再次执行简单插入排序。

以一种规则减少 gapgapgap 的大小，当 gapgapgap 为 1 时即简单插入排序，因此希尔排序又称作 缩小增量排序。

希尔在论文中提出的增量序列生成式为 n/2kn / 2^kn/2k，k=1,2,3,...k = 1, 2, 3, ...k=1,2,3,... ，例如 n=11n = 11n=11，则增量序列为 {1,2,5}\{1,2,5\}{1,2,5} 。在讨论希尔排序时，可将其称为 Shell 增量，另有更优的 Hibbard 增量、Knuth 增量、Sedgewick 增量等。

开始时 gapgapgap 较大，待排元素较少，因此排序速度较快。当 gapgapgap 较小时，基于第一点，此时待排序列已大致有序，排序效率接近线性复杂度。因此能够期待希尔排序复杂度将优于 O(n2)O(n^2)O(n2)。

# 冒泡排序

# 算法描述

从第一位开始从前往后比较相邻两个数字，若前者大，则交换两数字位置，然后比较位向右移动一位。第 1 轮从前到后的比较将使得最大的数字 冒泡 到最后，此时可以说一个数字已经被排序。每一轮的比较将使得当前未排序数字中的最大者被排序，未排序数字总数减 1。第 arr.length−1arr.length - 1arr.length−1 轮结束后排序完成。

# 优化

提前结束优化

当某一轮比较均未发生交换，说明排序已完成，可设置一个布尔值记录一轮排序是否有发生交换，若无则提前退出循环结束程序。

冒泡界优化

记录前一轮交换的最终位置，说明该位置之后的元素为已排序状态，下一轮的交换只需执行到该处。

public int[] bubbleSort(int[] arr) {
    if (arr.length < 2) return arr;
    for (int i = 0; i < arr.length - 1; i++) {
        boolean swapped = false;
        for (int j = 1; j < arr.length - i; j++) {
            if (arr[j - 1] > arr[j]) {
                swap(arr, j - 1, j);
                swapped = true;
            }
        }
        if (!swapped) break;
    }
    return arr;
}

public int[] bubbleSort(int[] arr) {
    if (arr.length < 2) return arr;
    boolean swapped = true;
    int lastSwappedIdx = arr.length - 1 ; // lastSwappedIdx 表示前一轮交换的最终位置
    while (swapped) {
        swapped = false;
        for (int i = 0; i < lastSwappedIdx; i++) {
            if (arr[i] > arr[i + 1]) {
                swap(arr, i, i + 1);
                swapped = true;
                swappedIdx = i;
            }
        }
        lastSwappedIdx = swappedIdx;
    }
    return arr;
}

# 快速排序

# 算法描述

快速排序是一种利用 分治思想 的排序方法，确定 主轴及分区 是快速排序的核心操作。首先在数组中确定一个主轴元素（将下标记为 pivotpivotpivot），然后将数组分为两部分，小于主轴的放在（确定最终位置的）主轴左侧，大于等于主轴的放在主轴右侧。递归地对主轴左右两侧数组执行这个过程，每次递归都传入待排序数组 arrarrarr 和本次要处理的部分的左右界，只处理这个范围内的序列。当所有递归都到达基准情形时，排序完成。因为是原地交换，递归过程中 arrarrarr 总是在动态排序，递归过程无需返回，为尾递归形式。

# 主轴的选择

主轴为起始元素 (quickSortSimplequickSortSimplequickSortSimple)。每次选取当前数组第一个元素作为主轴。

  • 优点：实现简单。
  • 缺点：若输入是较为有序的数组，pivotpivotpivot 总是不能均匀地分割数组。且若输入数组本身有序，简单快速排序将退化为冒泡排序，时间复杂度退化到 O(n2)O(n^2)O(n2)。

主轴为随机下标元素（quickSortRandomquickSortRandomquickSortRandom）。每次随机选取当前数组的下标，将该下标元素作为主轴。

  • 优点：避免了主轴为起始元素时对于基本有序的输入，因不能均匀分割数组导致复杂度退化的情况。
  • 缺点：随机数的选取本身消耗一定的性能。

主轴为左中右三数大小居中者（quickSortMedian3quickSortMedian3quickSortMedian3）。每次比较当前数组起始、中间和末尾三个元素的大小，选择大小居中者为主轴。

  • 优点：实现相对简单，且有效避免简单快速排序中的 劣质分割。
  • 缺点：三数取中方法消耗一定性能。

// 将左、中、右三数中值放到 left 位置上
private void median3(int[]arr, int l, int r) {
    int c = l + (r - l) / 2;
    if (arr[l] > arr[c]) swap(arr, l, c); // 左中，大者居中
    if (arr[c] > arr[r]) swap(arr, c, r); // 中右，大者居右，此时最大者居右
    if (arr[c] > arr[l]) swap(arr, l, c); // 左中，大者居左，此时中者居左
}

private int partition(int[] arr, int left, int right) {
    int pivot = left, index = pivot + 1;
    // 注意此时 right 是坐标，要执行到最后一个元素
    for (int i = index; i <= right; i++) {
        if (arr[i] < arr[pivot]) {
            swap(arr, index, i);
            index++;
        }
    }
    // 最后一个小于主轴元素的元素下标是 index - 1
    swap(arr, pivot, index - 1);
    return index - 1;
}

public int[] quickSortStack(int[] arr) {
    // 保存区间左右边界的栈，按 right 到 left 的顺序将初始区间界入栈
    Deque<Integer> stack = new ArrayDeque<>();
    stack.push(arr.length - 1);
    stack.push(0);
    // 判断栈是否空，不空则弹出一对 left,right 界
    while (!stack.isEmpty()) {
        int left = stack.pop(), right = stack.pop();
        if (left < right) {
            // 对 [left, right] 区间执行 partition 方法，得到 pivot
            /* 加入后续两行实现随机轴快排
             * int randomIndex = new Random().nextInt(right - left + 1) + left;
             * swap(arr, left, randomIndex);
             */
            /* 加入下行实现三数取中快排
             * median3(arr, left, right);
             */
            int pivot = partition(arr, left, right);
            // 当前 pivot 的左区间存在则将该区间 right,left 界入栈
            if (pivot > left) {
                stack.push(pivot - 1);
                stack.push(left);
            }
            // 当前 pivot 的右区间存在则将该区间 right,left 界入栈
            if (right > pivot) {
                stack.push(right);
                stack.push(pivot + 1);
            }
        }
    }
    return arr;
}

/*
 *     区间1               区间2                         区间3
 * +------------------------------------------------------------+
 * |  < pivot1  | pivot1 <= && <= pivot2  |    ?    | > pivot2  |
 * +------------------------------------------------------------+
 *              ^                         ^         ^
 *              |                         |         |
 *            lower                     index      upper
 */
private void dualPivotQuickSort(int[] arr, int left, int right) {
    if (left < right) {
        // 令左右两端元素中较小者居左，以 left 为初始 pivot1，right 为初始 pivot2
        // 即 arr[left] 为选定的左轴值，arr[right] 为选定的右轴值
        if (arr[left] > arr[right]) {
            swap(arr, left, right);
        }
        int index = left + 1; // 当前考察元素下标
        int lower = left + 1; // 推进到 pivot1 最终位置的下标，[left, lower) 确定在区间 1 中
        int upper = right - 1; // 推进到 pivot2 最终位置的下标，(upper, right] 确定在区间 3 中
        // [lower, index) 确定在区间 2 中，[index, upper] 为待考察区间

        // upper 以右（不含 upper）的元素都是确定在区间 3 的元素，所以考察元素的右界是 upper
        while (index <= upper) {
            // 若 arr[index] 小于左轴值，则 arr[index] 位于区间 1
            if (arr[index] < arr[left]) {
                // 交换 arr[index] 和 arr[lower]
                swap(arr, index, lower);
                // lower++，扩展区间 1，lower 位置向右一位靠近 pivot1 的最终位置
                lower++;
            }
            // 若 arr[index] 大于右轴值，则 arr[index] 位于区间 3
            else if (arr[index] > arr[right]) {
                // 先扩展区间 3，使得如下 while 结束后 upper 以右（不含 upper）的元素都位于区间 3
                // 区间 3 左扩不可使 index == upper
                // 否则之后的 upper-- 将导致 upper 为一个已经确定了区间归属的元素的位置
                while (arr[upper] > arr[right] && index < upper) {
                    upper--;
                }
                // 交换 arr[index] 和 arr[upper]
                swap(arr, index, upper);
                // upper--，扩展区间 3，upper 位置向左一位靠近 pivot2 的最终位置
                upper--;
                // 此时的 arr[index] 确定不在区间 3 中，但在区间 1 还是区间 2 中仍需明确
                // 若 arr[index] 小于左轴值，则 arr[index] 位于区间 1
                if (arr[index] < arr[left]) {
                    // 交换 arr[index] 和 arr[lower]
                    swap(arr, index, lower);
                    // lower++，扩展区间 1，lower 位置向右一位靠近 pivot1 的最终位置
                    lower++;
                }
            }
            index++; // 考察下一个数字
        }
        --lower; // 最后一个确定在区间 1 的元素下标是 --lower
        ++upper; // 最后一个确定在区间 3 的元素下标是 ++upper
        // 双轴归位，此时的 lower、upper 即分别为最终 pivot1、最终 pivot2
        swap(arr, left, lower);
        swap(arr, upper, right);
        // 对三个子区间分别执行双轴快排
        dualPivotQuickSort(arr, left, lower - 1); 		// 区间 1
        dualPivotQuickSort(arr, lower + 1, upper - 1); 	// 区间 2
        dualPivotQuickSort(arr, upper + 1, right); 		// 区间 3
    }
}

# 双元选择优化

在遍历寻找最小值下标 minIdxminIdxminIdx 时，可以同时寻找最大值下标 maxIdxmaxIdxmaxIdx ，这样一轮遍历可以确定两个元素的位置，遍历次数减少一半，但每轮次的操作变多，因此该优化只能少量提升选择排序的速度（复杂度介于单元选择排序复杂度及其一半之间，只有系数上的区别）。

public int[] selectSort(int[] arr) {

    for (int i = 0; i < arr.length - 1; i++) {

        for (int j = i + 1; j < arr.length; j++) {

public int[] selectSortDouble(int[] arr) {
    if (arr.length < 2) return arr;
    int n = arr.length;
    // 每轮确定两个数字，因此界会动态变化
    for (int i = 0; i < n - 1 - i; i++) {
        int minIdx = i, maxIdx = i;
        // 找到本轮执行中最小和最大的元素
        for (int j = i + 1; j < n - i; j++) {
            if (arr[j] < arr[minIdx]) minIdx = j;
            if (arr[j] > arr[maxIdx]) maxIdx = j;
        }
        // 若本轮最大值等于最小值，则未排序部分所有元素相等，排序完成
        if (minIdx == maxIdx) break;
        // 若本轮第一个数字不是最小值，则将最小值与本轮第一个数字交换位置
        if (minIdx != i) swap(arr, i, minIdx);
        // 交换 i 和 minIdx 时，有可能出现 i 即 maxIdx 的情况
        // 此时需要使 maxIdx = minIdx
        if (maxIdx == i) maxIdx = minIdx;
        // 若本轮最后一个数字不是最大值，则将最大值与本轮最后一个数字交换位置
        if (maxIdx != n - 1 - i) swap(arr, n - 1 - i, maxIdx);
    }
    return arr;
}

# 堆排序

# 堆的概念

堆（heap）可以定义为一棵二又树，树的节点中包含键（每个节点一个键），并且满足下面两个条件（这里仅讨论最大堆）：

  • 树的形状要求 —— 这棵二又树是 基本完备（essentially complete）的（或者简称为完全二又树），这意味着，树的每一层都是满的，除了最后一层最右边的元素有可能缺位。
  • 父母优势要求，又称为 堆特性（heap property）—— 每一个节点的键都大于或等于它子女的键（对于任何叶子我们认为这个条件都是自动满足的）。

# 堆的重要特性

不难证明：

  • 一棵存在 nnn 个节点的完全二叉树的高度等于 ⌊log⁡2n⌋⌊ \log_2n ⌋⌊log2​n⌋。
  • 我们可以通过数组来实现堆，从上到下、从左到右的方式来记录堆的元素。父节点的键位于数组 0∼⌊n/2⌋−10 \sim ⌊n/2⌋-10∼⌊n/2⌋−1 位置，而叶子节点的键位于数组 ⌊n/2⌋∼n−1⌊n/2⌋ \sim n-1⌊n/2⌋∼n−1 的位置。

  • 如果该节点不满足，把节点的键 KKK 和它子女的最大键进行交换，然后再检查在新位置上，KKK 是否满足父母优势要求。这个过程一直继续到对 KKK 的父母优势要求满足为止。

private void merge(int[] arr, int leftPos, int leftEnd, int rightEnd) {
    int i = leftPos, j = leftEnd + 1;

        while(i < j && arr[i] <= arr[j]) i++;

private void exchange(int[] arr, int left, int leftEnd, int rightEnd) {

    reverse(arr, leftEnd + 1, rightEnd);
    reverse(arr, left, rightEnd);
}

private void reverse(int[] arr, int start, int end) {
    while(start < end) {
        swap(arr, start, end);
        start++;
        end--;

# 桶排序

# 算法描述

桶排序是一种用空间换取时间的排序，桶排序重要的是它的思想，而不是具体实现，最佳时间复杂度可能是 O(n)O(n)O(n)，桶排序不是基于比较的排序而是一种分配式的。

桶排序的思想为：将待排序的序列分到若干个桶中，每个桶内的元素再进行单独排序。

桶排序选择的方案跟具体的数据有关系，桶排序是一个比较广泛的概念，计数排序、基数排序都是建立在桶排序的基础上。

当数据分布均匀且每个桶元素趋近 1 个时，时间复杂度能达到 O(n)O(n)O(n)。

public int[] bucketSort(int[] arr) {
    int n = arr.length;
    int max = arr[0], min = arr[0];
    for (int i = 1; i < n; i++) {
        if (arr[i] > max) max = arr[i];
        if (arr[i] < min) min = arr[i];
    }
    int bucketNum = (max - min) / n + 1;
    List<List<Integer>> bucketList = new ArrayList<>();
    for (int i = 0; i < bucketNum; i++) {
        bucketList.add(new ArrayList<>());
    }
    for (int i = 0; i < n; i++) {
        int bucketIdx = (arr[i] - min) / n;
        bucketList.get(bucketIdx).add(arr[i]);
    }
    // 桶内排序
    for (List<Integer> bucket : bucketList) {
        Collections.sort(bucket);
    }
    // 输出
    int index = 0;
    for (List<Integer> bucket : bucketList) {
        for (Integer integer : bucket) {
            arr[index] = integer;
            index++;
        }
    }
    return arr;
}

public int[] radixSort(int[] arr) {
    int n = arr.length;
    int max = Math.abs(arr[0]);
    // 找到 arr 中绝对值最大者
    for (int i = 1; i < n; i++) {
        max = Math.max(max, Math.abs(arr[i]));
    }
    int maxDigit = 0, exp = 1;
    while (max != 0) {
        maxDigit++;
        max /= 10;
    }
    // 接下来的 for 中，每轮都对当前位执行一次计数排序
    int[] sortedArr = new int[n];
    for (int i = 0; i < maxDigit; i++) {
        int[] countArr = new int[19];  // 处理负数优化
        // 根据每一个数字当前位的数字，累计相应位置的计数
        for (int j = 0; j < n; j++) {
            // 处理负数优化 +9
            int digit = (arr[j] / exp) % 10 + 9;
            countArr[digit]++;
        }
        // 计数数组变形
        for (int j = 1; j < countArr.length; j++) {
            countArr[j] += countArr[j - 1];
        }
        // 逆序输出保持稳定性
        for (int j = n - 1; j >= 0; j--) {
            int digit = (arr[j] / exp) % 10 + 9;
            sortedArr[countArr[digit] - 1] = arr[j];
            countArr[digit]--;
        }
        // 完成当前位的计数排序后将排序结果拷贝回原数组
        System.arraycopy(sortedArr, 0, arr, 0, n);
        // 准备下一轮对下一位的计数排序
        exp *= 10;
    }
    return arr;
}

稳定计数排序

public int[] countSort(int[] arr) {
    if (arr.length < 2) return arr;
    int n = arr.length, min = arr[0], max = arr[0];
    for (int i = 1; i < n; i++) {
        min = Math.min(min, arr[i]);
        max = Math.max(max, arr[i]);
    }
    int[] countArr = new int[max - min + 1];
    for (int i = 0; i < n; i++) {
        countArr[arr[i] - min]++;
    }
    for (int i = 1; i < countArr.length; i++) {  // 前缀和
        countArr[i] += countArr[i - 1];
    }
    int[] sortedArr = new int[n];
    for (int i = n - 1; i >= 0; i--) {
        sortedArr[countArr[arr[i] - min] - 1] = arr[i];
        countArr[arr[i] - min]--;
    }
    return sortedArr;
}
