# 逆康托展开

给定集合 [1,2,3,...,n][1, 2, 3, ..., n][1,2,3,...,n]，其全部元素有 n!n!n! 种排列。从小到大列出所有排列，并一一编号。求第 kkk 个排列。

对于给定的 nnn 和 kkk，我们不妨从左往右确定第 kkk 个排列中的每一个位置上的元素到底是什么。

我们首先确定排列中的首个元素 a1a_1a1​。我们知道：

  • 以 111 为 a1a_1a1​ 的排列一共有 (n−1)!(n-1)!(n−1)! 个；
  • 以 222 为 a1a_1a1​ 的排列一共有 (n−1)!(n-1)!(n−1)! 个；
  • ⋯\cdots⋯
  • 以 nnn 为 a1a_1a1​ 的排列一共有 (n−1)!(n-1)!(n−1)! 个。

我们需要求出从小到大的第 kkk 个排列，因此：

  • 如果 k≤(n−1)!k \leq (n-1)!k≤(n−1)!，我们就可以确定排列的首个元素为 111；
  • 如果 (n−1)!<k≤2⋅(n−1)!(n-1)! < k \leq 2 \cdot (n-1)!(n−1)!<k≤2⋅(n−1)!，我们就可以确定排列的首个元素为 222；
  • ⋯\cdots⋯
  • 如果 (n−1)⋅(n−1)!<k≤n⋅(n−1)!(n-1) \cdot (n-1)! < k \leq n \cdot (n-1)!(n−1)⋅(n−1)!<k≤n⋅(n−1)!，我们就可以确定排列的首个元素为 nnn。

因此，第 kkk 个排列的首个元素是：

a1=⌊k−1(n−1)!⌋+1 a_1 = \lfloor \frac{k-1}{(n-1)!} \rfloor + 1 a1​=⌊(n−1)!k−1​⌋+1

其中 ⌊x⌋\lfloor x \rfloor⌊x⌋ 表示将 xxx 向下取整。

当确定了 a1a_1a1​ 后，如何确定下一个元素 a2a_2a2​ 呢？实际上，考虑以 a1a_1a1​ 为首个元素的所有排列：

  • 以 [1,n]\a1[1,n] \backslash a_1[1,n]\a1​ 中的最小元素为 a2a_2a2​ 的排列一共有 (n−2)!(n-2)!(n−2)! 个；
  • 以 [1,n]\a1[1,n] \backslash a_1[1,n]\a1​ 中的次小元素为 a2a_2a2​ 的排列一共有 (n−2)!(n-2)!(n−2)! 个；
  • ⋯\cdots⋯
  • 以 [1,n]\a1[1,n] \backslash a_1[1,n]\a1​ 中的最大元素为 a2a_2a2​ 的排列一共有 (n−2)!(n-2)!(n−2)! 个。

其中 [1,n]\a1[1,n] \backslash a_1[1,n]\a1​ 表示包含 1,2,⋯n1, 2, \cdots n1,2,⋯n 中除去 a1a_1a1​ 以外元素的集合。这些排列从编号 (a1−1)⋅(n−1)!(a_1-1) \cdot (n-1)!(a1​−1)⋅(n−1)! 开始，到 a1⋅(n−1)!a_1 \cdot (n-1)!a1​⋅(n−1)! 结束，总计 (n−1)!(n-1)!(n−1)! 个。因此，第 kkk 个排列实际上就对应着 [1,n]\a1[1,n] \backslash a_1[1,n]\a1​ 的第

k′=(k−1)mod(n−1)!+1 k' = (k-1) \bmod (n-1)! + 1 k′=(k−1)mod(n−1)!+1

个排列。这样一来，我们把原问题转化成了一个完全相同但规模减少 111 的子问题：求 [1,n]\a1[1, n] \backslash a_1[1,n]\a1​ 这 n−1n-1n−1 个元素组成的排列中，第 k′k'k′ 小的排列。

算法思路

设第 kkk 个排列为 a1,a2,⋯,ana_1, a_2, \cdots, a_na1​,a2​,⋯,an​，从左往右地确定每一个元素 aia_iai​。

从小到大枚举 iii：

  • 当前已经使用过了 i−1i-1i−1 个元素，剩余 n−i+1n-i+1n−i+1 个元素未使用过，每一个元素作为 aia_iai​ 都对应着 (n−i)!(n-i)!(n−i)! 个排列，总计 (n−i+1)!(n-i+1)!(n−i+1)! 个排列；
  • 因此在第 kkk 个排列中，aia_iai​ 即为剩余未使用过的元素中第 ⌊k−1(n−i)!⌋+1\lfloor \frac{k-1}{(n-i)!} \rfloor + 1⌊(n−i)!k−1​⌋+1 小的元素；
  • 在确定了 aia_iai​ 后，这 n−i+1n-i+1n−i+1 个元素的第 kkk 个排列，就等于 aia_iai​ 之后跟着剩余 n−in-in−i 个元素的第 (k−1)mod(n−i)!+1(k-1) \bmod (n-i)! + 1(k−1)mod(n−i)!+1 个排列。
