# 2.1 ArrayList

# 属性与底层数据结构

// 默认初始容量
private static final int DEFAULT_CAPACITY = 10;
// 空数组，用于初始赋值
private static final Object[] EMPTY_ELEMENTDATA = {};
// 用于空 ArrayList 的共享空数组实例
private static final Object[] DEFAULTCAPACITY_EMPTY_ELEMENTDATA = {};
// 存储元素的数组
transient Object[] elementData;
// ArrayList的元素个数
private int size;

# 构造方法

// 构造函数
public ArrayList(int initialCapacity) {
    if (initialCapacity > 0) {
        this.elementData = new Object[initialCapacity];
    } else if (initialCapacity == 0) {
        this.elementData = EMPTY_ELEMENTDATA;
    } else {
        throw new IllegalArgumentException("Illegal Capacity: "+
                                           initialCapacity);
    }
}

// 默认构造函数，初始化一个空ArrayList
public ArrayList() {
    this.elementData = DEFAULTCAPACITY_EMPTY_ELEMENTDATA;
}

// 拷贝构造函数
public ArrayList(Collection<? extends E> c) {
    elementData = c.toArray();
    if ((size = elementData.length) != 0) {
        // 如果拷贝的集合不为空，则复制到 ArrayList 中
        if (elementData.getClass() != Object[].class)
            elementData = Arrays.copyOf(elementData, size, Object[].class);
    } else {
        // 如果拷贝的集合为空，则初始化一个空数组
        this.elementData = EMPTY_ELEMENTDATA;
    }
}

# 添加、获取、设置、删除元素

// 添加元素
public boolean add(E e) {
    ensureCapacityInternal(size + 1);  // 确保容量足够
    elementData[size++] = e;
    return true;
}

// 根据索引获取元素
@SuppressWarnings("unchecked")
public E get(int index) {
    rangeCheck(index);  // 检查索引范围
    return (E) elementData[index];
}

// 根据索引设置元素
public E set(int index, E element) {
    rangeCheck(index);  // 检查索引范围
    E oldValue = elementData(index);
    elementData[index] = element;
    return oldValue;
}

// 根据索引删除元素
public E remove(int index) {
    rangeCheck(index);  // 检查索引范围
    modCount++;  // 修改次数加1
    E oldValue = elementData(index);

    int numMoved = size - index - 1;
    if (numMoved > 0)
        System.arraycopy(elementData, index+1, elementData, index,
                         numMoved);
    elementData[--size] = null; // 清空末尾元素
    return oldValue;
}

# 扩容机制

// 确保容量足够
private void ensureCapacityInternal(int minCapacity) {
    if (elementData == DEFAULTCAPACITY_EMPTY_ELEMENTDATA) {
        minCapacity = Math.max(DEFAULT_CAPACITY, minCapacity);
    }

    ensureExplicitCapacity(minCapacity);
}

// 确保显式容量足够
private void ensureExplicitCapacity(int minCapacity) {
    modCount++;  // 修改次数加1
    if (minCapacity - elementData.length > 0)
        grow(minCapacity);  // 扩容
}

// 扩容
private void grow(int minCapacity) {
    int oldCapacity = elementData.length;
    int newCapacity = oldCapacity + (oldCapacity >> 1);
    if (newCapacity - minCapacity < 0)
        newCapacity = minCapacity;
    if (newCapacity - Integer.MAX_VALUE > 0)
        newCapacity = hugeCapacity(minCapacity);
    elementData = Arrays.copyOf(elementData, newCapacity);
}

# 2.2 LinkedList

# Node 类

LinkedList 的元素存储在 Node 类中，该类包含了元素值、前一个节点和后一个节点的引用。

具体定义如下：

private static class Node<E> {
    E item;
    Node<E> next;
    Node<E> prev;

    Node(Node<E> prev, E element, Node<E> next) {
        this.item = element;
        this.next = next;
        this.prev = prev;
    }
}

其中，prev 表示前一个节点的引用，next 表示后一个节点的引用，item 表示当前节点的值。

# 构造方法

LinkedList 有三个构造方法，分别是无参构造方法、以集合为参数的构造方法和以另一个 LinkedList 为参数的构造方法。

public LinkedList() {
}

public LinkedList(Collection<? extends E> c) {
    this();
    addAll(c);
}

public LinkedList(LinkedList<? extends E> c) {
    header = trailer = new Node<>(null, null, null);
    addAll(c);
}

# 添加元素

LinkedList 可以通过 add() 方法、addFirst() 方法和 addLast() 方法添加元素。add() 方法会将元素添加到链表的末尾，addFirst() 方法会将元素添加到链表的头部，addLast() 方法将元素添加到链表的末尾。

public boolean add(E e) {
    linkLast(e);
    return true;
}

public void addFirst(E e) {
    linkFirst(e);
}

public void addLast(E e) {
    linkLast(e);
}

其中，linkFirst() 方法会将元素添加到链表的头部，linkLast() 方法会将元素添加到链表的末尾。具体实现如下：

void linkFirst(E e) {
    final Node<E> f = header;
    final Node<E> newNode = new Node<>(null, e, f);
    header = newNode;
    if (f == null)
        trailer = newNode;
    else
        f.prev = newNode;
    size++;
}

void linkLast(E e) {
    final Node<E> l = trailer;
    final Node<E> newNode = new Node<>(l, e, null);
    trailer = newNode;
    if (l == null)
        header = newNode;
    else
        l.next = newNode;
    size++;
}

# 获取元素

LinkedList 可以通过 get() 方法、getFirst() 方法和 getLast() 方法获取元素。get() 方法会根据索引获取元素，getFirst() 方法会获取链表的头部元素，getLast() 方法会获取链表的尾部元素。具体代码如下：

public E get(int index) {
    checkElementIndex(index);
    return node(index).item;
}

public E getFirst() {
    final Node<E> f = header;
    if (f == null)
        throw new NoSuchElementException();
    return f.item;
}

public E getLast() {
    final Node<E> l = trailer;
    if (l == null)
        throw new NoSuchElementException();
    return l.item;
}

其中，node() 方法会根据索引获取对应的节点，代码如下：

Node<E> node(int index) {
    // assert isElementIndex(index);

    if (index < (size >> 1)) {
        Node<E> x = header;
        for (int i = 0; i < index; i++)
            x = x.next;
        return x;
    } else {
        Node<E> x = trailer;
        for (int i = size - 1; i > index; i--)
            x = x.prev;
        return x;
    }
}

其中，size >> 1 是对 size 进行移位操作，相当于除以 2，这里是为了判断索引位于前半段还是后半段，从而选择从头部还是尾部开始遍历。

# 删除元素

删除指定位置的元素

通过 remove(int index) 方法删除指定位置的元素，代码如下：

public E remove(int index) {
    checkElementIndex(index);
    return unlink(node(index));
}

E unlink(Node<E> x) {
    final E element = x.item;
    final Node<E> next = x.next;
    final Node<E> prev = x.prev;

    if (prev == null) {
        header = next;
    } else {
        prev.next = next;
        x.prev = null;
    }

    if (next == null) {
        trailer = prev;
    } else {
        next.prev = prev;
        x.next = null;
    }

    x.item = null;
    size--;
    return element;
}

先通过 node(int index) 方法获取要删除的节点，然后调用 unlink(Node<E> x) 方法进行删除。在 unlink(Node<E> x) 方法中，会将要删除节点的前驱节点和后继节点连接起来，从而删除该节点。

删除指定元素

通过 remove(Object o) 方法删除指定元素，代码如下：

public boolean remove(Object o) {
    if (o == null) {
        for (Node<E> x = header; x != null; x = x.next) {
            if (x.item == null) {
                unlink(x);
                return true;
            }
        }
    } else {
        for (Node<E> x = header; x != null; x = x.next) {
            if (o.equals(x.item)) {
                unlink(x);
                return true;
            }
        }
    }
    return false;
}

先判断要删除的元素是否为 null，如果是，则遍历整个链表，找到第一个 item 为 null 的节点并删除。如果要删除的元素不为 null，则遍历整个链表，找到第一个 item 等于该元素的节点并删除。

删除头部和尾部元素

通过 removeFirst() 方法和 removeLast() 方法删除头部和尾部元素，代码如下：

public E removeFirst() {
    final Node<E> f = header;
    if (f == null)
        throw new NoSuchElementException();
    return unlinkFirst(f);
}

public E removeLast() {
    final Node<E> l = trailer;
    if (l == null)
        throw new NoSuchElementException();
    return unlinkLast(l);
}

在删除头部和尾部元素时，分别调用 unlinkFirst(Node<E> f) 方法和 unlinkLast(Node<E> l) 方法进行删除。在这两个方法中，会将头节点和尾节点重新指向要删除节点的后继节点和前驱节点。
